# 🌍 Project Overview

A powerful desktop-based **Airline Scheduling & Cargo Management System** developed using **Python Tkinter**.

This system is designed to automate airline logistics by managing:

✅ Flights  
✅ Cargo Loads  
✅ Capacity Utilization  
✅ Profit Maximization  
✅ Smart Scheduling  
✅ Interactive Dashboard  
✅ AI Chatbot Support

---

# 🚀 Main Features

## ✈️ Flight Module

- Add new flights  
- Set capacity  
- View remaining space  
- Track active flights  

## 📦 Cargo Module

- Add cargo shipments  
- Define weight  
- Define profit value  
- Manage cargo queue  

## ⚡ Smart Scheduler

- Automatically assigns cargo  
- Prioritizes high-profit cargo  
- Optimizes available flight capacity  
- Displays scheduling results instantly  

## 📊 Dashboard Analytics

- Total Flights  
- Total Cargo  
- Assigned Cargo  
- Total Revenue
